Welcome to the TWC Cybersecurity Virtual Escape Room 2021!

The game is simple; we have constructed a series of challenges that you need to complete before the time runs out.

Rules

1. Each team must pick a leader. The leader must be able to take control of the session in teams and be able to describe what is happening on the screen to team members.

2. The leader can be changed at any time.

3. The game begins once the timer starts.

4. Participants may use any external resources available to them, including Google.

5. At any time, you can ask for a "CLUE"; however, using a clue will cause your team to incur a penalty.

6. During each event, there will be two hosts from cybersecurity available to answer questions and help with technical issues.

Notes

· Each task requires a passcode.

· Passcodes are found by finishing the puzzle.

· The game ends when you enter the correct code.

· Please let the cybersecurity hosts know if an accommodation will be needed.

Your first task is to find and unscramble the hidden word. Clicking the hidden word will take you to Task 2.

Before starting the game, let us know to start the timer.

Instructions for setting the timer

Rules

  • You must press the correct code before the timer ends
  • A wrong code will result in a 25% reduction of the time
  • If the countdown hits 00:00:00, the timer ends

Stopping the Timer

  1. Press the correct code, which consist out of 4 to 6 numbers and/or letters
  2. Press # to confirm
  3. If correct, the screen will show "ESCAPED"

You can arm the timer ONLY when it's not armed yet (obviously). To do so, follow the following steps:

  1. The screen will show "UNARMED"
  2. Press A * * * (so A and 3x *)
  3. The screen will show "SET CODE"
  4. Give a code of 4 to 8 numbers or letters
  5. Press # to confirm
  6. The screen will show "SET TIME"
  7. Give a time of 1 to 6 numbers, in HHMMSS format
  8. Press # to confirm
  9. The screen will show "READY"
  10. Press A # to arm the timer
  11. The screen will now show "ARMED" and a countdown will start running
